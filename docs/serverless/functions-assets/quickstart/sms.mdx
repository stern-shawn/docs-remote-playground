For the most minimal possible example, one can reply to an incoming SMS with a hardcoded message. To do so, [create a new MessagingResponse](focus://3) and [declare the intended message contents](focus://5). Once your message content has been set, you can return the generated TwiML by [passing it to the callback function](focus://8) as shown and signaling a successful end to the Function.

## Respond to an inbound SMS

```js
exports.handler = (context, event, callback) => {
  // Create a new messaging response object
  const twiml = new Twilio.twiml.MessagingResponse();
  // Use any of the Node.js SDK methods, such as `message`, to compose a response
  twiml.message('Hello, World!');
  // Return the TwiML as the second argument to `callback`
  // This will render the response as XML in reply to the webhook request
  return callback(null, twiml);
};
```
